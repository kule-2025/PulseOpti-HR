## 平台概述

PulseOpti HR 脉策聚效是一个功能强大的人力资源管理 SaaS 平台，提供全方位的企业人力资源解决方案，包括员工管理、招聘管理、绩效管理、薪酬管理、考勤管理等核心功能。

### 主要特性

- **员工全生命周期管理** - 从入职到离职的全流程管理
- **数据同步机制** - 支持类似钉钉的便捷数据同步
- **智能告警系统** - 实时监控，及时预警
- **多租户架构** - 支持企业独立数据隔离
- **权限管理** - 细粒度的角色权限控制
- **AI 智能分析** - 数据驱动的决策支持

## 核心功能模块

### 1. 员工管理

#### 1.1 创建员工档案

**操作步骤：**

1. 进入「员工管理」模块
2. 点击「添加员工」按钮
3. 填写员工基本信息：
   - 姓名（必填）
   - 邮箱（必填，用于登录账号）
   - 电话（必填）
   - 性别、出生日期等
4. 填写职位信息：
   - 所属部门（必填）
   - 职位名称（必填）
   - 职级
   - 用工类型
5. 填写合同信息：
   - 入职日期（必填）
   - 试用期结束日期
   - 合同类型
6. 填写联系方式与紧急联系人
7. 点击「保存」完成创建

**注意事项：**

- 员工邮箱必须唯一，系统会自动创建登录账号
- 带有「必填」标记的字段必须填写
- 上传头像前请确保图片符合格式要求

#### 1.2 查看员工列表

**操作步骤：**

1. 进入「员工管理」模块
2. 默认显示所有在职员工
3. 使用搜索框快速查找员工：
   - 支持按姓名、邮箱、电话搜索
4. 使用筛选功能：
   - 按部门筛选
   - 按职位筛选
   - 按在职状态筛选
5. 查看员工详细信息：
   - 点击员工姓名进入详情页

#### 1.3 编辑员工信息

**操作步骤：**

1. 进入员工详情页
2. 点击「编辑」按钮
3. 修改需要更新的信息
4. 点击「保存」确认修改

#### 1.4 员工状态管理

支持对员工进行以下操作：

- **转正** - 试用期员工转正
- **晋升** - 调整员工职级
- **调岗** - 调整员工部门或职位
- **离职** - 办理离职手续
- **复职** - 已离职员工重新入职

### 2. 招聘管理

#### 2.1 发布职位

**操作步骤：**

1. 进入「招聘管理」模块
2. 点击「发布职位」
3. 填写职位信息：
   - 职位名称
   - 所属部门
   - 工作地点
   - 薪资范围
   - 职位描述
   - 职位要求
4. 设置招聘流程
5. 发布职位

#### 2.2 简历管理

**操作步骤：**

1. 进入「简历管理」
2. 支持以下操作：
   - 上传简历
   - 邀请面试
   - 标记面试状态
   - 录用/拒绝
   - 移入人才库

#### 2.3 面试安排

**操作步骤：**

1. 在简历列表选择候选人
2. 点击「安排面试」
3. 选择面试官
4. 设置面试时间
5. 发送面试通知

### 3. 绩效管理

#### 3.1 创建考核周期

**操作步骤：**

1. 进入「绩效管理」模块
2. 点击「新建考核」
3. 设置考核周期
4. 选择参与考核的员工
5. 设置考核指标
6. 发布考核

#### 3.2 员工自评

**操作步骤：**

1. 进入「我的考核」
2. 选择待考核周期
3. 填写自评内容
4. 上传相关证明材料
5. 提交自评

#### 3.3 上级评分

**操作步骤：**

1. 进入「团队考核」
2. 选择需要评分的员工
3. 查看员工自评内容
4. 进行评分和评价
5. 提交评分结果

### 4. 薪酬管理

#### 4.1 设置薪资结构

**操作步骤：**

1. 进入「薪酬管理」模块
2. 进入「薪资设置」
3. 创建薪资项目：
   - 基本工资
   - 绩效奖金
   - 津贴补贴
   - 加班工资
4. 设置计算规则
5. 保存薪资结构

#### 4.2 生成薪资单

**操作步骤：**

1. 进入「薪资管理」
2. 选择薪资周期
3. 点击「生成薪资单」
4. 系统自动计算：
   - 应发工资
   - 社保公积金
   - 个人所得税
   - 实发工资
5. 确认生成

#### 4.3 薪资发放

**操作步骤：**

1. 查看待发放薪资单
2. 核对数据无误
3. 点击「确认发放」
4. 生成发放记录
5. 员工收到薪资通知

### 5. 考勤管理

#### 5.1 排班管理

**操作步骤：**

1. 进入「考勤管理」模块
2. 进入「排班设置」
3. 选择日期
4. 设置班次：
   - 工作日班次
   - 周末班次
   - 节假日安排
5. 批量应用到员工

#### 5.2 打卡记录

**操作步骤：**

1. 进入「打卡管理」
2. 查看员工打卡记录：
   - 上班打卡
   - 下班打卡
   - 外勤打卡
3. 导出打卡数据

#### 5.3 请假审批

**操作步骤：**

1. 进入「请假管理」
2. 查看待审批申请
3. 点击申请查看详情
4. 批准或拒绝
5. 添加审批意见

## 数据同步管理

### 功能概述

数据同步模块提供类似钉钉的便捷数据同步功能，支持全量同步、增量同步和实时同步。

### 1. 创建同步任务

**操作步骤：**

1. 进入「数据同步」模块
2. 点击「新建同步任务」
3. 选择同步类型：
   - **全量同步** - 同步所有数据
   - **增量同步** - 仅同步变化的数据
   - **实时同步** - 实时监听并同步数据
4. 选择同步范围：
   - 用户
   - 部门
   - 职位
   - 员工
   - 全部
5. 设置优先级（1-10，数字越大优先级越高）
6. 选择执行方式：
   - 立即执行
   - 定时执行
7. 点击「创建」完成任务创建

### 2. 监控同步任务

**操作步骤：**

1. 进入「数据同步」模块
2. 查看「同步任务列表」
3. 任务状态说明：
   - **待执行** - 任务已创建，等待执行
   - **执行中** - 任务正在执行
   - **已完成** - 任务执行成功
   - **已失败** - 任务执行失败
4. 查看任务详情：
   - 同步进度
   - 同步记录数
   - 执行时间

### 3. 查看同步日志

**操作步骤：**

1. 选择一个同步任务
2. 点击「查看日志」
3. 日志级别说明：
   - **info** - 一般信息
   - **warn** - 警告信息
   - **error** - 错误信息

### 4. 任务操作

**重试失败任务：**

1. 找到失败的任务
2. 点击「重试」按钮
3. 任务重新加入执行队列

**取消任务：**

1. 找到待执行或执行中的任务
2. 点击「取消」按钮
3. 确认取消操作

### 5. 同步统计

**查看统计信息：**

1. 进入「数据同步」模块
2. 查看「统计仪表盘」
3. 显示内容：
   - 总任务数
   - 成功任务数
   - 失败任务数
   - 执行中任务数
   - 各类型同步成功率

## 告警管理

### 功能概述

告警系统实时监控数据同步状态，当出现异常时自动触发告警，帮助管理员及时处理问题。

### 1. 查看活跃告警

**操作步骤：**

1. 点击顶部通知图标
2. 查看「告警」标签页
3. 活跃告警按严重程度排序：
   - **critical** - 严重（红色）
   - **high** - 高（橙色）
   - **medium** - 中（黄色）
   - **low** - 低（蓝色）

### 2. 确认告警

**操作步骤：**

1. 打开一个告警详情
2. 查看告警信息：
   - 告警标题
   - 告警描述
   - 相关任务
   - 触发时间
3. 点击「确认」按钮
4. 添加处理说明（可选）

### 3. 告警规则管理

**查看规则：**

1. 进入「系统设置」
2. 选择「告警规则」
3. 查看所有告警规则

**新增规则：**

1. 点击「添加规则」
2. 选择规则类型：
   - 同步失败告警
   - 同步超时告警
   - 错误阈值告警
3. 设置触发条件
4. 选择通知方式：
   - 仪表盘通知
   - 邮件通知
   - 短信通知
5. 添加接收人
6. 保存规则

**修改规则：**

1. 找到需要修改的规则
2. 点击「编辑」
3. 修改规则配置
4. 保存更改

**启用/禁用规则：**

1. 找到目标规则
2. 点击开关按钮
3. 确认操作

## 常见问题

### Q1: 如何重置员工密码？

**答：** 管理员可以进入员工详情页，点击「重置密码」按钮，系统会生成临时密码并通过邮件发送给员工。

### Q2: 同步任务失败怎么办？

**答：**
1. 查看「数据同步」模块中的失败任务
2. 点击「查看日志」了解失败原因
3. 修复问题后，点击「重试」按钮重新执行

### Q3: 如何批量导入员工？

**答：**
1. 进入「员工管理」模块
2. 点击「批量导入」
3. 下载导入模板
4. 按模板格式填写员工信息
5. 上传文件并确认导入

### Q4: 薪资计算有误如何处理？

**答：**
1. 检查薪资项目设置是否正确
2. 检查员工考勤数据是否完整
3. 进入薪资详情页
4. 点击「重新计算」
5. 确认数据后重新生成

### Q5: 如何设置审批流程？

**答：**
1. 进入「系统设置」
2. 选择「工作流设置」
3. 创建新的审批流程
4. 设置审批节点和审批人
5. 启用流程

### Q6: 告警通知如何配置？

**答：**
1. 进入「系统设置」
2. 选择「告警规则」
3. 编辑对应规则
4. 勾选需要的通知方式
5. 添加接收人
