## 系统简介

PulseOpti HR 脉策聚效是一款智能化的人力资源管理SaaS平台，整合了组织人事、招聘管理、绩效管理、薪酬管理、考勤管理、培训管理、离职管理等全方位HR功能，并融入AI智能分析、工作流引擎、数据同步等高级特性，帮助企业提升人力资源管理效率。

### 核心特性

- **智能化管理**: AI驱动的员工分析、绩效预测、离职预警
- **全流程覆盖**: 从入职到离职的全生命周期管理
- **灵活配置**: 支持自定义工作流、权限配置、审批流程
- **数据驱动**: 实时统计分析、智能告警、决策支持
- **安全可靠**: 多租户隔离、RBAC权限、审计日志

## 快速开始

### 1. 注册和登录

访问平台首页，点击"注册"按钮，填写以下信息：

- 企业名称
- 管理员邮箱
- 管理员密码
- 验证码

注册成功后，使用邮箱和密码登录系统。

### 2. 初始化设置

首次登录后，需要完成以下初始化设置：

1. **完善企业信息**
   - 企业logo
   - 企业地址
   - 联系方式
   - 业务描述

2. **组织架构设置**
   - 创建部门
   - 设置职位
   - 配置职级体系

3. **用户管理**
   - 添加员工
   - 分配角色
   - 设置权限

## 核心功能

### 1. 组织人事

#### 员工管理

**功能说明**: 管理员工档案、组织架构、职位体系

**操作步骤**:

1. 进入"组织人事" -> "员工管理"
2. 点击"添加员工"按钮
3. 填写员工基本信息
   - 姓名
   - 工号
   - 邮箱
   - 手机号
   - 部门
   - 职位
   - 入职日期
   - 薪资
4. 点击"保存"完成添加

**高级功能**:
- 批量导入员工
- 员工档案编辑
- 员工离职办理
- 员工转岗调薪

#### 组织架构

**功能说明**: 管理企业组织架构，支持多层级部门

**操作步骤**:

1. 进入"组织人事" -> "组织架构"
2. 点击"添加部门"按钮
3. 填写部门信息
   - 部门名称
   - 上级部门
   - 部门负责人
   - 部门描述
4. 点击"保存"完成添加

**高级功能**:
- 可视化组织架构图
- 部门人员统计
- 部门权限配置

### 2. 招聘管理

#### 岗位发布

**功能说明**: 发布和管理招聘职位

**操作步骤**:

1. 进入"招聘管理" -> "岗位发布"
2. 点击"发布职位"按钮
3. 填写职位信息
   - 职位名称
   - 所属部门
   - 工作地点
   - 薪资范围
   - 职位描述
   - 任职要求
4. 点击"发布"完成发布

**高级功能**:
- AI职位描述生成
- 职位状态管理
- 职位推广设置

#### 简历管理

**功能说明**: 管理候选人简历，支持AI简历解析

**操作步骤**:

1. 进入"招聘管理" -> "简历管理"
2. 点击"上传简历"按钮
3. 选择简历文件（支持PDF、Word格式）
4. 系统自动解析简历信息
5. 审核简历内容
6. 添加到人才库或安排面试

**高级功能**:
- AI简历解析
- 简历去重检测
- 简历评分筛选

#### 智能面试

**功能说明**: AI驱动的智能面试系统

**操作步骤**:

1. 进入"招聘管理" -> "智能面试"
2. 选择候选人
3. 设置面试题目（AI自动生成或自定义）
4. 开始面试
5. 查看AI面试评分和报告

**高级功能**:
- AI面试题目生成
- 实时面试评分
- 面试报告生成

### 3. 绩效管理

#### 目标设定

**功能说明**: 设定员工绩效目标

**操作步骤**:

1. 进入"绩效管理" -> "目标设定"
2. 选择员工
3. 点击"添加目标"按钮
4. 填写目标信息
   - 目标名称
   - 目标描述
   - 权重
   - 完成标准
   - 截止日期
5. 点击"保存"完成设定

#### 绩效评估

**功能说明**: 对员工绩效进行评估

**操作步骤**:

1. 进入"绩效管理" -> "绩效评估"
2. 选择评估周期
3. 查看员工目标完成情况
4. 填写评估意见和评分
5. 提交评估结果

**高级功能**:
- AI绩效预测
- 绩效趋势分析
- 绩效对比分析

### 4. AI智能分析

#### 员工分析

**功能说明**: AI驱动的员工数据分析

**操作步骤**:

1. 进入"统计分析" -> "AI智能分析"
2. 点击"开始分析"按钮
3. 系统自动生成分析报告，包括：
   - 员工结构分析
   - 人效分析
   - 薪资分析
   - 离职风险分析
4. 查看详细报告和建议

#### AI问答

**功能说明**: 向AI提问任何HR相关问题

**操作步骤**:

1. 进入"统计分析" -> "AI问答"
2. 在输入框中输入问题
3. 点击"提问"按钮
4. 查看AI的回答和建议

**问题示例**:
- "如何降低员工离职率？"
- "招聘渠道效果如何？"
- "绩效评估有哪些改进建议？"
- "薪酬水平是否合理？"

### 5. 工作流引擎

#### 流程定义

**功能说明**: 定义和管理工作流

**操作步骤**:

1. 进入"工作流" -> "流程定义"
2. 点击"创建流程"按钮
3. 选择流程模板（入职、离职、晋升、转岗、调薪等）
4. 配置流程节点
   - 审批节点
   - 通知节点
   - 条件节点
5. 设置审批人和规则
6. 保存流程定义

#### 流程发起

**功能说明**: 发起工作流程

**操作步骤**:

1. 在相关业务页面点击"发起审批"
2. 选择工作流程
3. 填写申请信息
4. 提交申请
5. 查看审批进度

### 6. 告警监控

#### 系统告警

**功能说明**: 监控系统告警信息

**操作步骤**:

1. 进入"告警监控"页面
2. 查看活跃告警列表
3. 点击"确认"按钮处理告警
4. 查看告警详情和历史

**告警类型**:
- 系统异常
- 性能警告
- 数据异常
- 安全告警

### 7. 数据同步

#### 同步任务

**功能说明**: 管理数据同步任务

**操作步骤**:

1. 进入"数据同步"页面
2. 查看同步任务列表
3. 创建新的同步任务
4. 配置同步规则
5. 启动同步任务

**同步类型**:
- 全量同步
- 增量同步
- 实时同步

## 常见问题

### Q1: 如何重置密码？

**A**:
1. 点击登录页面的"忘记密码"
2. 输入注册邮箱
3. 查收邮件中的重置链接
4. 设置新密码

### Q2: 如何添加子账号？

**A**:
1. 进入"设置" -> "账号管理"
2. 点击"添加子账号"
3. 填写账号信息
4. 分配角色和权限
5. 保存设置

### Q3: 如何配置审批流程？

**A**:
1. 进入"工作流" -> "流程定义"
2. 创建或编辑流程
3. 添加审批节点
4. 设置审批人
5. 保存流程

### Q4: AI分析的准确性如何？

**A**:
AI分析基于大数据和机器学习算法，能够提供有价值的参考建议。但请结合实际情况，AI分析结果仅供参考，最终决策请以实际情况为准。

### Q5: 数据安全如何保障？

**A**:
- 数据传输采用SSL加密
- 数据库定期备份
- 多租户数据隔离
- 完善的权限控制
- 审计日志记录
