## 🎯 一、高级AI功能增强

### 1.1 AI简历智能解析（深度）

**功能描述**:
- AI自动解析简历中的关键信息（姓名、联系方式、教育经历、工作经历、技能、项目经验）
- 智能提取工作成果和业绩数据
- 自动生成候选人画像和标签
- 简历相似度分析和重复检测

**页面路径**: `/dashboard/recruiting/ai-resume-parser`

**核心功能**:
- ⏳ 支持多种简历格式（PDF、Word、HTML、图片OCR）
- ⏳ AI字段提取（20+字段）
- ⏳ 智能标签生成（技能标签、潜力标签）
- ⏳ 简历重复检测（基于相似度算法）
- ⏳ 批量解析（支持上传多份简历）

**API端点**:
```
POST /api/ai/resume-parse          - 解析单份简历
POST /api/ai/resume-batch-parse    - 批量解析简历
GET  /api/ai/resume-duplicate      - 检测重复简历
```

**与超管端交互**:
- 超管端可查看AI解析统计（解析次数、准确率、节省时间）
- 超管端可配置AI解析参数（提取字段、标签规则）
- 超管端可导出解析数据报表

**闭环实现**:
1. 用户上传简历 → AI解析 → 提取信息 → 自动填充候选人表单
2. 批量解析 → 批量上传 → 并行处理 → 批量创建候选人记录
3. 重复检测 → 解析后检测 → 提示用户是否合并 → 更新候选人库

**工作流引擎集成**:
- 与招聘工作流集成：解析后的简历直接进入筛选流程
- 与人才库集成：自动将解析后的候选人加入人才库

**AI模型**: 豆包大语言模型 + OCR模型 + 相似度算法

**预计开发时间**: 8小时

**优先级**: P1

```
POST /api/integrations/feishu/sso              - SSO登录
POST /api/integrations/feishu/sync-org         - 同步组织架构
```
